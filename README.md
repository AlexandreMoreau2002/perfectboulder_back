# Backend API

Minimal FastAPI backend with Docker support.

## Getting Started

1. Start the project:
   ```bash
   make up
   ```

2. Access the API:
   - Health check: http://localhost:8000/health
   - Docs: http://localhost:8000/docs

3. Other commands:
   - `make logs`: View logs
   - `make down`: Stop containers
